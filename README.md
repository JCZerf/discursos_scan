# discursos_scan

Um projeto simples para escanear, visualizar e gerenciar discursos.

## Estrutura do Projeto

- `index.html`: Página principal do projeto.
- `assets/`: Pasta para recursos estáticos.
- `component/`: Componentes e dados, incluindo:
  - `discursos_data.json`: Base de dados dos discursos.
  - `homepage.html`: Página inicial dos componentes.
- `css/`: Arquivos de estilos (`scanner.css`, `styles.css`).
- `js/`: Scripts JavaScript do projeto (`menu.js`, `scanner.js`).
  - `users.json`: Dados de usuários.

## Como usar

1. Abra o arquivo `index.html` em um navegador web.
2. Navegue pelas funcionalidades usando o menu.
3. Os dados dos discursos e usuários estão em arquivos JSON na pasta `component/` e `js/`.

## Requisitos

- Navegador moderno (Chrome, Firefox, Edge, etc.)
